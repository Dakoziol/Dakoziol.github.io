Game Development in Unity Engine

I have been dabbling with Unity for a few years now. In fact, I built a mobile game for my high school senior project titled Kayak-O-Rama!, a 2-D, top-down, local multiplayer kayak racing game. I even introduced myself to Git via Bitbucket and SourceTree for this project, but I still have a lot to learn with that. I’ve since started a few other projects, all very physics based, but I haven’t finished one yet. My web Authoring final project is also relevant as a showcase for Kayak-O-Rama! and one of my later projects.
